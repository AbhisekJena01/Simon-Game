About the Game

Firstly, the game shows the first colour in the sequence ( For eg :- blue). The user clicks on that coloured button.

Next, the game shows the next colour (For eg :- red), the user has to remember the sequence of colours (i.e. blue, red) and so on and so forth.

If the user messes up the sequence, then the game ends.

Controls

Press on the button i.e. the coloured space or you can press the number key associated with the color

1 - Green

2 - Red

3 - Yellow

4 - Blue

Click here or press Enter to return to the Game.
